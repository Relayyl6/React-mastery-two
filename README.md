## How to Use This Repository

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/react-learning.git
   ```

2. Navigate to specific project/examples:
   ```bash
   cd project-folder
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Run the development server:
   ```bash
   npm start
   ```
